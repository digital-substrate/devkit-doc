Ecosystem
=========

A technical orientation across the dsviper ecosystem: which component does
what, and what depends on what.

Components are designed so each can stand on its own, but they compose into
two value chains that together form the full DevKit experience. If you are
new, start with :doc:`value-chains` — it is the map.
